# <img src="assets/images/covid19-logo.png" width="50px" height="50px">[COVID19 US Specific Dashboard](https://ncov19.us/)

## App Summary
Dashboard where users can view trend data & statistics for the COVID19 pandemic in the U.S. Data describing the number of confirmed cases, deaths, and recovered patients is available for the whole country and by state as well. News updates for the outbreak are displayed in a newsfeed from major outlets and tweets from public officials and institutions. Additional information about the virus, symptoms, and resources are available as well.

View the app  [here](https://ncov19.us/).


## Tech Stack
- Plotly Dash
- MongoDB
- Twitter API
